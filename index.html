Naughty or Nice?

This map shows if a country is generally being Naughty or Nice about Christmas at any given moment. Naughtiness or niceness is loosely determined based on an analysis of keywords used in tweets. The realtime data comes is streaming from the Twitter Streaming API and through Pusher using a Pusher/Twitter API integration app and into your browser. Importantly, only data that your client needs is sent to the web browser. This demo is inspired by one originally created by @girlie_mac.
